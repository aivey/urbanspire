Turkish Belly Dancing    Group

Class Description

This is where all new dancers start. This class gives you the understanding of basic belly dance technique, posturing, language, move execution and it helps you to build necessary strength. You must have a working knowledge of the material in the intro class to move into the intermediate drills. Moving into the intermediate drills class must be approved by your instructor. This class is drop in friendly.
